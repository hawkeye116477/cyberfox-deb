Installing

Method 1

Download and install deb packages from here.

Method 2

Run these commands to add this repostory to your APT sources list: 

$ echo "deb https://hawkeye116477.github.io/cyberfox-deb release main" | \
sudo tee /etc/apt/sources.list.d/cyberfox-release-hawkeye116477.list

And the public GPG key: 

$ curl https://hawkeye116477.github.io/cyberfox-deb/hawkeye116477.key | 
sudo apt-key add -

You can download and install the packages after you update the APT package index: 

$ sudo apt-get update

Note: Language packs and KDE integration features are available as separate packages to install from apt repository!
